### Datasets
For The Bible Romani-Romanian dataset use the `dataset_romani.json` and `dataset_romanian.json` files.
For the dataset parsed from the `Suport-De-Curs-Limba-Romani.pdf` (Ghearghe Sarau) file use the `sentences_romani_romanian.txt` and `vocabulary_romani_romanian.txt`.
